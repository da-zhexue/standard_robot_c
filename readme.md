# 嵌入式机器人代码框架  

## 概述  
该代码是由本人设计的RoboMaster竞赛的嵌入式软件框架，基于STM32 HAL库与FreeRTOS实时操作系统构建。它采用BSP、Module、APP三层架构设计，旨在实现硬件驱动、功能模块与上层应用的解耦，提升代码的复用性、可维护性与不同机器人平台间移植效率。
## 项目框架  

| 层级 | 名称 | 主要内容                            |  
| ------- | ------- |---------------------------------|
| BSP | 硬件支持包 | USART、CAN、PWM等外设在HAL库基础上进一步封装   |
| Module | 外设模块层 | 继承自BSP类，实现如具体外设功能如上位机通信协议、电机控制等 |
| APP | 应用层 | 整合各模块，实现机器人底盘、云台等功能             |
  
## 框架目标
### 统一代码库
通过完善 BSP 与提供丰富的 Module，建立团队共享的代码框架。
### 降低移植成本
当更换机器人兵种或发生硬件改动时，通常只需修改或增补 App 层业务逻辑及少量外设模块，核心驱动与通用模块无需重写。
### 提升开发效率
电控同学可将精力集中于机器人策略与性能优化，而非重复编写底层驱动。
